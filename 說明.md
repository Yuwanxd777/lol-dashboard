# LOL 數據儀表板

Oracle's Elixir 官方資料 → 本機全自動儀表板。開 `index.html` 就能用。

## 資料流（v3，已脫離 Google Sheets）

```
OE 官方公開 Drive 資料夾（2014~2026 各一份 CSV，官方每日更新）
  → fetch_data.py（service account 下載＋本機重算 RAW_DATA）
  → data_{年}.js ＋ data.js（年份清單）
  → index.html（所有統計都在瀏覽器現算）
```

- **每日排程** `LOL_Dashboard_Update`（10:00）只重抓今年；歷史年份有檔就跳過
- 手動更新：雙擊 `update.bat`；強制重抓某年：`python fetch_data.py 2025`；全部：`--force`
- 需求：GCP 專案 `mslol-500204` 已開通 Drive API（2026-07-07 完成）

## 分頁

| 分頁 | 內容 |
|---|---|
| 總覽 | 藍紅方勝率、場均時長、選手/戰隊評分 Top5、英雄 Presence Top10、最近比賽 |
| 選手 | 25 欄完整數據（含 K/D/A、CS差@10）、下路組/中野組雙人模式、搜尋、**點選手名→英雄池** |
| 戰隊 | 29 欄（含藍/紅方勝率、首殺）、對戰模式、**點戰隊名→BP紀錄** |
| 英雄 | P+B%（Presence）、勝-敗、參與率、CS差、**點英雄→版本趨勢折線** |
| 對戰BP | 統計條（藍紅勝率/場均時長）、時長、聯賽色籤、頭像 BP、選角順位、⭐先選 |
| AI 問答 | 讀當前篩選數據作答、每題 NT$ 計價（匯率每日自動）、200 題歷史紀錄 |

## 視覺規則

- **台灣習慣：越高越紅、越低越綠**（評分徽章、排行圖）
- 表格：每欄**前三紅（第一粗體）、後三藍（最後粗體）**；名次金/銀/銅；前兩欄凍結
- 全部選項（篩選、年份、分頁、雷達軸 3~12 邊、對比人選）自動記憶
- 右上 🖊️ 畫筆：講解模式，筆跡跟頁面捲動

## 評分公式（復刻自原 GS 公式）

加權百分位 →曲線 (1-(1-x/100)^1.5) → 場數懲罰 → **評分＝攻擊×0.7＋防守×0.3**。
英雄評分＝掌握度(Wilson×3+KDA×2+DPM+贏線)70%＋版本強度(Ban×2+金差)30%。

## 疑難排解

- 「尚未產生資料」→ 跑 `update.bat`，看 `update_log.txt`
- 403 Drive API → GCP 開通 Drive API（見上）
- 英雄圖破圖 → Data Dragon 版本自動更新，個別新英雄檔名對不到會顯示文字
